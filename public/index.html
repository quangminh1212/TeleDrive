Hướng dẫn đồng bộ lên Telegram

File của bạn đã được lưu trữ cục bộ. Để đồng bộ với Telegram, vui lòng thực hiện các bước sau:

  1. Mở Telegram Web trong tab mới
  2. Đăng nhập vào tài khoản Telegram của bạn
  3. Chọn Saved Messages (tin nhắn đã lưu) hoặc chat/channel mong muốn
  4. Nhấn vào biểu tượng (đính kèm)
  5. Tải lên file từ thư mục: uploads/
Để tự động hóa quá trình này, bạn có thể cấu hình BOT_TOKEN trong file .env
