Cấu hình Telegram API

Cập nhật thông tin API cho TeleDrive

Để sử dụng TeleDrive, bạn cần cung cấp thông tin API Telegram:

1
Truy cập trang đăng ký API Telegram

Mở my.telegram.org/auth và đăng nhập vào tài khoản Telegram của bạn.

2
Tạo ứng dụng mới

Chọn "API development tools" và điền thông tin ứng dụng của bạn:

  • App title: TeleDrive (hoặc tên tùy chọn)
  • Short name: TeleDrive (chỉ chữ cái, không dấu)
  • Các trường khác: tùy chọn
3
Lấy thông tin API

Sau khi tạo ứng dụng, bạn sẽ nhận được:

  • api_id: Một dãy số
  • api_hash: Một chuỗi ký tự
4
Nhập thông tin API vào form bên dưới
